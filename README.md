# Programmers Guild of Lausanne

I use this repository to familiarise myself with fundamental computer science concepts and math problems often posed in data science job interviews. Many of these problems and topics have been brought up in the Programmers Guild of Lausanne that is a weekly meetup event for data scientists and coding enthusiasts.

## Requirements

- Python 3.x
- Jupyter (optional, for interactive notebooks)

## 📁 Project Structure

- **binary_trees/** - Binary tree implementations and examples
  - `binary_tree.py` - Core binary tree node class
  - `binary_tree.ipynb` - Jupyter notebook with interactive examples
  - `binary_tree_example1.png` - Visual representation of tree structure

- **search_algorithms/** - Search algorithm implementations
  - `binary_search.py` - Binary search implementation
  - `sorting_algorithms.py` - Various sorting algorithms

- **sorting_algorithms/** - Sorting algorithm implementations
  - `sorting_algorithms.py` - Various sorting algorithms
